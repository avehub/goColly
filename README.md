# 汽车、摩托车数据爬取
## 介绍
该Spider是对多个领域产品信息进行整合的数据平台，包括汽车、摩托、自行车、相机、穿戴设备等多种类型，同时满足了多个类型数据的采集、持久化、维护和管理等基础功能。

## 软件架构
Golang：GoColly
Database：Mysql、OSS
